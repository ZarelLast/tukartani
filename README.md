# 🌾 Tukar Tani

> **Game Web Simulasi Ekonomi:** Kelola keluarga petani kopi di Desa Eldoria, hadapi fluktuasi ekonomi global, dan taklukkan pergerakan kurs! 
> Dibuat khusus untuk **JuaraVibeCoding (#1 Game)**.

## 📖 Premis Permainan
Kamu adalah kepala keluarga petani kopi di **Desa Eldoria**. Hasil panen kopi kamu akan diekspor dan dibayar dengan mata uang global (**Gold Coins / GC**), tetapi untuk kebutuhan bertani (seperti pupuk dan BBM), kamu harus mengimpor dan membayarnya dengan mata uang lokal (**Selga**). 

Jantung dari permainan ini adalah pergerakan kurs yang tidak terduga. *Selga melemah?* Kopi kamu akan laku lebih mahal dalam Selga, tapi harga pupuk akan melambung tinggi! *Selga menguat?* Impor murah, tapi hasil ekspor kopi kamu turun! 📈📉

Atur *timing* jual & beli melawan pergerakan kurs, kelola asetmu, dan jaga kesejahteraan keluargamu!

## ✨ Fitur Utama
* **Dilema Ekonomi Nyata:** Merasakan langsung *trade-off* dalam mengambil keputusan jual beli (ekspor-impor) di tengah fluktuasi mata uang.
* **Integrasi AI yang Dinamis:** Narasi berita ekonomi yang di-generate oleh AI secara prosedural, plus fitur **Penasihat AI** interaktif (mendukung *Speech-to-Text* dan *Text-to-Speech*!). `[soon]`
* **Arsitektur Hybrid (Phaser JS + React):** Rendering dunia permainan menggunakan Phaser yang mulus di dalam `<canvas>`, dipadukan dengan DOM UI Overlay menggunakan React yang cantik dan responsif!
* **Deterministik RNG:** Anti *save-scumming*. Keputusan dan strategi menentukan hasil, bukan sekadar keberuntungan. 
* **Mobile-First Design:** Dimainkan nyaman di perangkat mobile, kapan saja dan di mana saja.

## 🛠️ Tech Stack
* **Frontend:** React + Tailwind CSS
* **Game Engine:** Phaser JS
* **Integrasi LLM:** API LLM Proxy
* **Data Visualization:** Recharts
